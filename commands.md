## Run A1, A2, A3 etc.
```
python3 simulator/simulator.py -c tests-configs/A1-config.json
python3 simulator/simulator.py -c tests-configs/A2-config.json
python3 simulator/simulator.py -c tests-configs/A3-config.json
```

## Run with verbose output - details exactly how revenues are computed
```
python3 simulator/simulator.py -c tests-configs/A1-config.json -v
```

## Run with extra info
```
python3 simulator/simulator.py -c tests-configs/A1-config.json i
```

## Run all tests
```
bash local-tester.sh
```
